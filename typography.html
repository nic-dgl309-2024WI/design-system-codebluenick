Font uses:

Element Font Family Font Size Font Weight Font Style Utility class
Body text Source Sans 3 16px 400 Regular u-ff-body
Headings Roboto 32px 700 Regular u-ff-heading
CTA Buttons Source Sans 3 14px 500 Regular u-ff-body

Utility class specification table:

Utility class Description Example
.u-ff-body Applies the font family (Source Sans 3) for body text Source Sans Text
.u-ff-heading Applies the font family (Roboto) for headings Roboto Text
.u-fs-heading Applies font size (32px) for headings Headings
.u-fs-body Applies font size (16px) for body text Body text
.u-fs-button Applies font size (14px) for buttons Button text
.u-fw-bold Applies font weight (700) for headings Headings
.u-fw-regular Applies font size (400) for body text Body text
.u-fw-medium Applies font size (500) for buttons Button text
.u-lh-1 Applies line height (1) Line Height
.u-lh-15 Applies line height (1.5) Line Height
.u-ls-1 Applies letter spacing (1px) Letter spacing
.u-ls-2 Applies letter spacing (2px) Letter spacing
